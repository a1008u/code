引数

Like · Reply · 3 hrs

通常の引数

function showMessage(value) {
 console.log(value);
}

showMessage();
showMessage('山田');
showMessage('山田', '鈴木');

特徴

  1. 与える引数の数が、関数側で要求する数と異なる場合も、引数チェックをしない
  2. showMessage('山田', '鈴木')の場合、argumentsオブジェクトで引数は格納されている

デフォルト引数

function showMessage2(value = 'test') {
 console.log(value);
}

showMessage2();
showMessage2('山田');
showMessage2('山田', '鈴木');

特徴

  1. 関数実行時に引数がない場合に評価されて、デフォルト値を代入する
  2. 引数がある場合は、引数の値を優先して利用する
