引数

Like · Reply · 3 hrs

通常の引数

function show(x, y = 1) {
 console.log('x = ' + x);
 console.log('y = ' + y);
}
show();

特徴

  1. 値が渡されなかったからといって、引数の不足を通知してくれるわけではないのです

デフォルト引数

function required() {
 throw new Error('引数が不足しています');
}
function hoge(value = required()) {
 return value;
}
hoge();

特徴

  1. 独自のrequiredメソッドの利用で対応する
