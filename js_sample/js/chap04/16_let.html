let

if (true) {
 let i = 5;
}

 console.log(i);

特徴

  1. let命令は、ブロックスコープに対応した変数宣言

letと即時実行関数

{
 let i = 5;
 console.log(i);
}

 console.log(i);

特徴

  1. let命令はブロックスコープに対応しているため、ブロックで囲めば即実行関数は不要

letとswitch

switch(x) {
 case 0:
  let value = 'x:0';
 case 1:
   let value = 'x:1';
}

特徴

  1. caseはラベルで修飾された句でありブロックでないため、重複となる
