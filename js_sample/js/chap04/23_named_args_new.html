名前付き引数

Like · Reply · 3 hrs

名前付き引数 {プロパティ名 = デフォルト値, ... }

function getTriangle({ base = 1, height = 1 }) {
 return base * height / 2;
}

const value = getTriangle({ base:5, height:4 });
console.log(value);

const value2 = getTriangle({});
console.log(value2);

特徴

  1. 省略可能な引数をスマートに表現できる
  2. 引数の順番を自由に変更できる
