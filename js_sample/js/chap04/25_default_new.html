  2. showMessage('山田', '鈴木')の場合、argumentsオブジェクトで引数は格納されている

デフォルト引数

function multi(a, b = a) {
 return a * b;
}
console.log(multi(10, 5));
console.log(multi(3));

特徴

  1. 関数実行時に引数がない場合に評価されて、デフォルト値を代入する
  2. 引数がある場合は、引数の値を優先して利用する

デフォルト引数

function multi2(a = b, b = 5) {
 return a * b;
}
console.log(multi2());

特徴

  1. ほかの引数をデフォルト値とする場合、参照できるのは、自身より前に定義されたものだけである点に注意してください

デフォルト引数

function getTriangle2(base = 1, height = 1) {
 return base * height / 2;
}
console.log(getTriangle2(5, null));

特徴

  1. デフォルト値が適用されるのは、引数が明示的に渡されなかった場合だけです。
  2. たとえばnull／false／0／空文字列など、意味的に空を表すような値でも、それらが明示的に渡された場合は、デフォルト値が適用されることはありません。

デフォルト引数

function getTriangle2(base = 1, height) {
 return base * height / 2;
}
console.log(getTriangle2(10));

特徴

  1. デフォルト値を持った仮引数は、引数リストの末尾で宣言すべきです
