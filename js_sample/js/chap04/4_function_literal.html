関数リテラル表現

Like · Reply · 3 hrs

書き方

let 変数名 = function(引数...){ 関数の本体 };

サンプル

let getTriangle = function(base, height) {
  return base * height / 2;
};
console.log('三角形の面積：' + getTriangle(5, 2));

特徴

  1. リテラルとして表現できる
  2. 関数リテラルを変数に代入したり、ある関数の引数として渡したり、はたまた、戻り値として関数を返すことすら可能
