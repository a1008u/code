関数リテラル表現

functionリテラル

this.data = 'this is within global';
 function func() {
  console.log(this.data);
 }
 const f = {
  data: 'within object',
  execute: func
 };
 f.execute();

アロー関数

this.data = 'this is within global';
 const arrowFunc = () => {
  console.log(this.data);
 };
 const af = {
  data: 'this is within object',
  execute: arrowFunc
 };
 af.execute();

特徴

  1. アロー関数式は既存の関数式より文字数が短くなるだけではなく、宣言時のthisを束縛して不変のものにするという効果を持っている
