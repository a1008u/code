function命令

Like · Reply · 3 hrs

書き方

function 関数名(引数, ...) {
  ...任意の処理...
  return 戻り値;
}

サンプル

function getTriangle(base, height) {
  return base * height / 2;
}

console.log('三角形の面積：' + getTriangle(5, 2));

特徴

  1. （単なる文字列ではなく）識別子の条件を満たす必要がある
  2. 「その関数がどのような処理を担っているのか」がすぐわかるような名前を付ける
    → 「showMessage」のように「動詞＋名詞」の形式で命名するのが一般的
